== Description ==

Woocommerce Payment Gateway.

Pay securely using your bank. (only available in New Zealand for BNZ and Westpac banks)

Avali is approved by leading banks as the safe, secure way to pay instantly online - direct from your bank account. There's no need to register or share your internet banking details.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Click 'Settings'
4. Select 'Checkout > Checkout options > Avali'
5. Enable the gateway
6. Enter your 'Avali Merchant Key', to obtain a merchant key, go here: https://www.avali.nz/contact/
